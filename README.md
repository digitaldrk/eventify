# Eventify

Client for Eventify. Allows to publish events from Ruby-applications.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'eventify'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install eventify
